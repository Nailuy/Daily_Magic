## 🔮 About

**Daily Magic** is an immersive quest platform built on Solana. It bridges the gap between community engagement and on-chain activity. Users connect their wallets to embark on a journey, completing daily rituals and milestones to earn XP and ascend through magical ranks.

> *From Verified Human to Grand Wizard — your journey begins here.*

---

## ⚡ Key Features

* **🔐 Seamless Web3 Auth** Instant login via Solana wallets (Phantom, Solflare) with secure session management.
    
* **⚔️ Dynamic Quest Engine** A robust system handling daily rotations, social verification, and on-chain actions with built-in fair play mechanics.

* **🏆 Live Progression System** Real-time XP tracking and automatic rank evolution based on user activity.

* **📊 Global Leaderboard** Competitive tracking of the top magic users in the ecosystem.

---

## 🛠️ Tech Stack

* **Core:** Next.js (App Router), TypeScript
* **Styling:** Tailwind CSS, Framer Motion
* **Backend:** Supabase (PostgreSQL + Edge Functions)
* **Blockchain:** @solana/web3.js

## 🚀 Getting Started

To run the project locally for development:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/your-username/daily-magic.git](https://github.com/your-username/daily-magic.git)
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Configure Environment**
    Create a `.env.local` file and add your project credentials.

4.  **Run the App**
    ```bash
    npm run dev
    ```
